This repository is a structured study resource for software design and development. It collects concept guides, Java implementations, UML-focused material, and question banks for exam preparation and technical review.

## How To Use

1. Start with the concept guides to review the main ideas and examples.
2. Read the Java implementations to connect each concept with code.
3. Practice with the MCQ, true/false, and scenario-based question banks.
4. Use the justifications to understand why each answer is correct.

## Purpose

The goal of this repository is to provide a clear, reusable, and well-organized study reference for software design and development topics through direct knowledge checks, practical code examples, and applied scenario analysis.
