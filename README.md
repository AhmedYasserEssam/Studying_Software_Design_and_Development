# Studying Software Design and Development

This repository is a structured study resource for software design and development topics. It is intended to collect organized revision material, question banks, and scenario-based practice questions that support exam preparation and conceptual review.

## Current Study Sections

### [SOLID Principles](SOLID_Principles_Guide.md)

This guide explains the five SOLID object-oriented design principles with clear descriptions, practical notes, and bad/good Java code examples.

### [Creational Design Patterns](creational_design_patterns)

This section includes Java implementations and a guide for creational design patterns:

- Singleton
- Factory Method
- Abstract Factory

The guide explains where each pattern appears in the code, when to use it, its main SOLID principle focus, and UML diagrams for the implementation.

Files:

- [Java implementation](creational_design_patterns/Main.java)
- [Creational design patterns guide](creational_design_patterns/Creational_Design_Patterns_Guide.md)

### [Structural Design Patterns](structural_design_patterns)

This section includes Java implementations and a guide for structural design patterns:

- Adapter
- Bridge
- Decorator
- Flyweight

The guide explains how these patterns organize relationships between classes and objects, with code examples, SOLID notes, and UML diagrams.

Files:

- [Java implementation](structural_design_patterns/Main.java)
- [Structural design patterns guide](structural_design_patterns/Structural_Design_Patterns_Guide.md)

### [Architectural Design](architectural_design)

This section focuses on architectural design concepts and includes:

- [150 multiple-choice questions with answers and justifications](architectural_design/architectural_design_mcq_question_bank.md)
- [250 true/false questions with answers and justifications](architectural_design/architectural_design_true_false_question_bank.md)

### [UML Component & Deployment Diagrams](Component%26Deployment_diagrams)

This section focuses on UML component and deployment diagrams and includes:

- [160 multiple-choice questions with answers and justifications](Component%26Deployment_diagrams/uml_component_deployment_mcq_question_bank.md)

## Purpose

The goal of this repository is to provide a clear, reusable, and well-organized study reference for reviewing software design and development topics through both direct knowledge checks and practical scenario analysis.
